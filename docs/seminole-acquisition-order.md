# Seminole Homeowner Acquisition Order

## Phase 1: Prove local conversion

1. Send every neighborhood, partner, flyer, and direct-message link to `/seminole` with a unique referral code, for example `/seminole?ref=ANGELO`.
2. Start with Seminole ZIP codes and neighborhood groups where FixMyHome already has a relationship or service-provider coverage.
3. Measure landing views, form starts, completed short forms, signup starts, and submitted jobs in `CampaignEvent`.
4. Review results weekly by referral code. Keep sources that produce submitted jobs, not merely clicks.

## Phase 2: Build repeatable referral loops

1. Give each active handyman, property manager, realtor, and neighborhood ambassador a stable referral code.
2. Ask newly served homeowners for a review and one neighbor referral after a completed job.
3. Create one focused seasonal offer: storm preparation, HVAC tune-ups, tree trimming, or common repair bundles.

## Phase 3: Add paid acquisition carefully

1. Retarget campaign visitors who started but did not finish the form.
2. Test search ads only for high-intent Seminole repair terms and send each ad group to a matching campaign variant.
3. Cap spend until cost per submitted job and the percentage of jobs receiving qualified bids are stable.

## Weekly scorecard

- Landing-to-form-start rate
- Form-start-to-completion rate
- Completion-to-signup rate
- Signup-to-job-submission rate
- Jobs receiving at least one qualified bid
- Cost per submitted job by source and referral code
