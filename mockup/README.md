# FixMyHome Mockup

Standalone review mockup for `fixmyhome.pro`.

Open `index.html` in a browser to review the prototype. The mockup includes:

- Homeowner dashboard
- Job posting form
- Bid comparison view with AI recommendation treatment
- Handyman dashboard
- Messaging thread

This folder is intentionally static HTML/CSS/JS so it can be reviewed without running the Next.js app or connecting to the database.
